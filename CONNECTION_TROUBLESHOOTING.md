## 🚨 常见问题

### 问题 1：云服务器安全组未开放端口

**症状**：浏览器提示 "无法访问此网站" 或 "连接超时"

**解决方案**：

需要在云服务商控制台配置安全组规则：

**腾讯云/华为云/阿里云**：

1. 登录云服务器控制台
2. 找到实例 → 安全组
3. 添加入站规则：
   - 协议：TCP
   - 端口：8888, 9999
   - 源：0.0.0.0/0（允许所有 IP）

**配置示例**：
```
方向：入站
协议：TCP
端口范围：8888,9999
授权对象：0.0.0.0/0
```

---

### 问题 2：服务器防火墙阻止

**症状**：服务在服务器上运行，但外部无法访问

**解决方案**：

检查并开放防火墙端口：

```bash
# 检查防火墙状态
sudo systemctl status firewalld
# 或
sudo ufw status

# 开放端口
sudo firewall-cmd --add-port=8888/tcp --permanent
sudo firewall-cmd --add-port=9999/tcp --permanent
sudo firewall-cmd --reload

# 或使用 ufw
sudo ufw allow 8888/tcp
sudo ufw allow 9999/tcp
```

---

### 问题 3：服务未监听所有网络接口

**症状**：只能从服务器本地访问，无法从外部访问

**检查**：

```bash
# 检查服务监听地址
netstat -tuln | grep -E "8888|9999"
```

**正确输出**：
```
tcp        0      0 0.0.0.0:8888            0.0.0.0:*               LISTEN
tcp        0      0 0.0.0.0:9999            0.0.0.0:*               LISTEN
```

如果显示 `127.0.0.1:8888` 而不是 `0.0.0.0:8888`，需要修改服务启动参数：

```bash
# 后端服务
uvicorn src.app:app --host 0.0.0.0 --port 8888

# 前端服务
python -m http.server 9999 --bind 0.0.0.0
```

### 问题 4：网络问题

**症状**：间歇性连接或速度慢

**解决方案**：

1. 检查本地网络是否正常
2. 尝试使用手机热点测试
3. 检查是否有代理或 VPN 阻止
4. 尝试其他浏览器
