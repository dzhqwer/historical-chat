## 📖 项目简介

### ✨ 核心特性

#### 🎭 多角色对话
- **预设人物**：李白、孔子、爱因斯坦、莎士比亚、苏格拉底、屈原、牛顿、达芬奇
- **角色扮演**：AI 完美模拟每个历史人物的性格、语言风格和时代背景
- **动态创建**：支持自动搜索并创建未预设的历史人物

#### 🗣️ 语音交互
- **语音合成（TTS）**：将文字转换为自然的语音
- **多音色支持**：每个历史人物配有专属音色

#### 🎨 3D 模型展示
- **AI 图像生成**：自动生成高质量历史人物画像
- **3D 渲染**：使用 Three.js 实现 3D 虚拟形象展示
- **动态效果**：浮动动画、说话动画、光晕效果

#### 👄 口型同步
- **音素生成**：基于文本自动生成口型序列
- **21 种音素**：支持丰富的口型变化

## 🛠️ 技术栈

### 后端
- **FastAPI** - 高性能 Web 框架
- **LangGraph** - Agent 框架
- **LangChain** - LLM 工具链
- **Python 3.10+**

### 前端
- **React** - UI 框架
- **Three.js** - 3D 渲染
- **Tailwind CSS** - 样式框架

### AI 能力
- **Doubao (豆包)** - 大语言模型
- **Coze SDK** - AI 能力集成
